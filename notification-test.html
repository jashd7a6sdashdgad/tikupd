üß™ How to Test

Follow these steps to test the Smart Notifications System:
  1. Start the development server: 
    npm run dev
  2. Navigate to any protected page (login required)
  3. Look for the notification bell in the top navigation
  4. Test notification creation:
    • Go to Settings page
    • Find the "Smart Notification Settings" section
    • Use the test buttons to create notifications
  5. Test VIP contacts:
    • Add VIP contacts in settings
    • Verify priority filtering works
  6. Test API endpoints: 
    # Create demo notifications
curl -X PATCH http://localhost:3000/api/notifications

# Test specific notification
curl -X POST http://localhost:3000/api/notifications \
  -H "Content-Type: application/json" \
  -d '{"action": "test", "data": {"type": "critical"}}'
